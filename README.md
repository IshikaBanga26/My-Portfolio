# Personal Portfolio – Ishika Banga

A modern developer portfolio built with **React, Tailwind CSS, and Vite** to showcase my projects, learning journey, and achievements in technology.

The portfolio highlights my work, skills, and milestones while providing an easy way for visitors to explore my projects and get in touch.

---

## Features

* Responsive and modern UI
* Hero section introducing my background
* About section describing my journey and interests
* Projects showcase with GitHub and live links
* Milestones section highlighting certifications and achievements
* Contact form integration using **EmailJS**
* Smooth animations and interactive UI elements

---

## Tech Stack

* **React** – Frontend library
* **Tailwind CSS** – Styling and responsive design
* **Vite** – Fast build tool
* **EmailJS** – Contact form email integration
* **Lucide Icons** – Modern icon library

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/IshikaBanga26/Ishika-Portfolio.git
```

### 2. Navigate to the project

```bash
cd ISHIKA-PORTFOLIO
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the development server

```bash
npm run dev
```
